Wes Anderson palettes for R

Wes Anderson palettes

A tumbler user set up a blog where he breaks down the colour palettes used in Wes Anderson movies. This by itself is ok, but then someone called Karthik Ram turned these palettes into ggplot2 palettes, and released it as a package called wesanderson on CRAN.

The package

First load the package, and get the available palettes. namelist tells us which movies have been added, and how many colours are in the palette.

1
2
3
library(wesanderson)
# Current list of available palettes
  namelist

The palettes

Code below loops through the list of palettes and plots them.

1
2
3
4
5
6
7
par(mfrow=c(3,3))
for (i in 1:nrow(namelist)){
  name_movie <- namelist[i,1]
  number_colours <- namelist[i,2]
  display.wes.palette(number_colours,
                      name_movie)
}
